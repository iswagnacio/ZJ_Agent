## Workflow Patterns

**Basic Cell Counting**:
1. job_00: Split channels (if needed)
2. job_01: Segment ROIs (detect cells/nuclei)
3. job_02: Create target from segmented ROIs
4. job_99: Formula to count target.COUNT

**Positive Rate Analysis** (e.g., Ki67 positive rate):
1. job_00: Split channels
2. job_01: Segment all nuclei (DAPI channel)
3. job_02: Create target "all_nuclei"
4. job_03: Measure Ki67 intensity in all nuclei
5. job_04: Create target "positive_nuclei" (filter by intensity)
6. job_99: Formula: positive_nuclei.COUNT / all_nuclei.COUNT

**Co-localization Analysis**:
1. job_00: Split channels
2. job_01: Segment objects in channel A
3. job_02: Segment objects in channel B
4. job_03: Create targets from both channels
5. job_04: Identify overlapping objects
6. job_99: Formula to calculate overlap percentage

Remember: Generate complete, valid JSON that can be directly parsed and executed.
